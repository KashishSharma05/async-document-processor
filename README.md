# 🚀 Async Document Processor (DocFlow)

A full-stack document processing system with real-time job tracking, modern SaaS UI, and asynchronous workflow simulation.

---

## 🔥 Features

- 📂 Upload documents with drag-and-drop
- ⚡ Asynchronous processing (Queued → Processing → Completed)
- 📊 Real-time dashboard with live status updates
- 🎯 Job tracking system with progress indicators
- 🎨 Premium UI with animations and modern design
- 🔔 Toast notifications for user actions
- 📈 Analytics cards with trends (+12%, +8%, etc.)

---

## 🛠️ Tech Stack

**Frontend**
- React.js
- Tailwind CSS
- Framer Motion

**Backend**
- Node.js
- Express.js

**Other**
- REST APIs
- Async job simulation
- Modern UI/UX design principles

## ⚙️ How It Works

1. User uploads a document
2. Job is added to queue
3. Backend simulates processing
4. Status updates in real-time
5. User can track progress visually
